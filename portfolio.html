Portfolio

Worlds of Fun LLC POS (point of sales)

Description:

I built and maintain the POS system for the Worlds of Fun LLC game and toy store. This system contains a fully function POS system with a cash drawer, receipt printer and credit card reader. The system allows for returns for store credit or cash. There is also a fully functional inventory management system that can run in parallel for live updates to the inventory. The inventory management also allows for items to be discounted for a specified period to take effect in the POS system. There is also an administrative website that allows for real time metrics of sales and current inventory. The system uses a SQL database to integrate all the data.

Technologies:

  • C#
  • WPF
  • SQL
  • MySQL
  • Python 3
  • Python-Django
